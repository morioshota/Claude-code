# 改修ロードマップ — 作業帯図作成ツール

正太との実務利用を前提に、今後の改修候補を「効果 × 難易度」で整理する。
上ほど優先度が高い（実務での効きが大きく、単一 HTML 制約を崩さない）。
各項目に **触る箇所** を添えてあるので、Claude Code はそこから着手できる。

## 優先度 高（次にやると効く）

### R1. 縮尺の実務精度アップ：縮尺バー / 縮尺値の直接入力
現状は 2 点+実距離のみ。図面に「1/500」等の縮尺表記で入れたい場面がある。
- 縮尺設定モーダルに「縮尺分母を直接入力（例 500）」タブを追加。用紙の物理寸法(A3=420mm 横)と PAPER.w から `mPerPx` を逆算。
- 図面枠内に**スケールバー**（0–5–10m 目盛）を描画するオプション。
- **触る箇所**: `finishCalib` 周辺、`drawSheet`（スケールバー描画追加）、`settings` に `scaleDenominator`。

### R2. 記号ラベル・引き出し線
標示板や看板に「工事件名」「連絡先」等の文字を紐づけたい。
- `symbol` に `label` を持たせ、任意方向へ引き出し線＋テキストを描画。
- **触る箇所**: `drawObj` の symbol 分岐、`renderProps`（label 入力欄）、データモデル。

### R3. 複数ページ PDF 対応 / ページ選択
現状は PDF の 1 ページ目固定。
- `loadBackground` でページ数を取得し、2 ページ以上ならページ選択モーダルを表示。
- **触る箇所**: `ensurePdfJs`/`loadBackground`、簡易ページ選択 UI。

### R4. 自動保存の堅牢化（背景画像の肥大対策）
背景 dataURL が大きいと localStorage 上限に当たり自動保存が黙って失敗する。
- 背景画像を JPEG 圧縮（画質 0.7 程度）してから保存、または IndexedDB へ退避。
- 保存失敗時に「JSON 保存を推奨」トーストを明示表示。
- **触る箇所**: `autosave`、`setBgFromDataURL`（保存用に圧縮 dataURL を別途生成）。

## 優先度 中（あると便利）

### R5. 自動配置パターンの拡充・調整
- 歩道規制、両側交互、片側 2 車線の 1 車線規制などパターン追加。
- 各パターンで「標示板の設置距離（手前 50/100/200m）」を反映した予告標示板の自動列置き。
- **触る箇所**: `runTemplate`、ウィザードモーダルの選択肢と入力欄。

### R6. スナップ機能
- 頂点・記号配置時に、近くの頂点／グリッド／背景の直線にスナップ。
- **触る箇所**: `pointerMove`（drag 中の座標補正）、スナップ判定ヘルパ追加。

### R7. 整列・分布・等間隔配置
- 選択した複数記号を一直線・等間隔に整える（コーンの手置き補助）。
- **前提**: まず**複数選択**の実装（現状は単一選択）。`state.sel` を配列化する設計変更を伴う。

### R8. 寸法線の一括・角度寸法
- 連続寸法（測点列）、角度寸法。すり付け勾配・幅員の記入を楽にする。
- **触る箇所**: `dim` 型の拡張または新 type、`drawDim`。

### R9. 図面レイアウトの調整
- 表題欄のカスタム（会社名・図番・日付・縮尺表記の欄）。
- 凡例の列数・文字サイズ・順序の手動調整。
- **触る箇所**: `drawSheet` の表題欄、`drawLegend`。

## 優先度 低（将来／要検討）

### R10. テンプレート保存
- よく使う規制構成を「マイテンプレート」として保存し、別図面に再利用。
- **触る箇所**: 選択オブジェクト群の JSON 化と localStorage 名前付き保存。

### R11. PDF 直接出力
- 現状は PNG＋印刷。jsPDF 等の遅延ロードで PDF 保存を追加（単一 HTML 制約は維持、CDN 遅延ロード）。
- 印刷経由の「PDF として保存」で足りるかは実運用で判断。

### R12. 座標・面積の表示
- 作業帯の面積(m²)、規制延長(m)の自動集計をパネルに表示。
- **触る箇所**: `renderProps` または footer、多角形面積計算ヘルパ。

---

## やらない / 慎重に

- **ラスタ地形図からの完全自動作図**：精度が許可申請水準に届かず、警察協議で通らない。半自動を維持する。
- **サーバー化・ビルド導入**：単一 HTML・依存なしの制約を壊すため不可。ライブラリは必ず CDN 遅延ロードで。
- **外部 API への図面送信**：申請図面は機微情報。既定でオフライン完結を守る。
