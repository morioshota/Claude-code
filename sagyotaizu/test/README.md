# ヘッドレス回帰チェック（開発用）

`作業帯図作成ツール.html` は単一HTML・依存なしの成果物ですが、ブラウザ無しでロジックの
回帰を素早く検知するための開発用チェックをここに置いています（成果物には含めません）。

```bash
cd sagyotaizu
node test/headless-check.cjs            # 隣の 作業帯図作成ツール.html を検証
node test/headless-check.cjs path.html  # 明示指定も可
```

- 最小限の DOM / Canvas2D シムで HTML 内の `<script>` を評価し、`init()` は実行せずに
  内部関数を直接 assert します（縮尺換算・描画経路の無例外・保存往復・選択/複数パターン/
  一括出力/整列/集計 など）。
- **見た目の最終確認はブラウザで**（`CODE_STRUCTURE.md` 末尾「テスト観点」）。この
  チェックは「壊れていないか」の一次防波堤です。
- 機能を追加したら、このファイル末尾のアサーション群にも1〜数項目足してください。
