# 作業帯図作成ツール — 仕様書 / README

道路使用許可申請用の「作業帯図」を、地形図(PDF/画像)を背景に半自動で作図し、手直し・出力できる単一 HTML ツール。

- **現行バージョン**: v1.1
- **成果物**: `作業帯図作成ツール.html`（単一ファイル・ビルド不要）
- **想定利用者**: 土木施工管理の実務者（正太）。社内 PC + デスクトップブラウザで使用。
- **設計上の絶対制約**: 外部ランタイム依存なしの単一 HTML。外部ライブラリ（PDF.js）は PDF 読込時のみ CDN から遅延ロード。ビルドステップ・サーバー不要でそのまま開ける。

---

## 1. これは何をするツールか / しないか

### する
- 地形図（PDF の 1 ページ目、または PNG/JPG 画像）を背景に読み込み、透明度調整
- 2 点クリック + 実距離入力で**縮尺を設定**（以後すべての寸法・間隔が実寸連動）
- 規制パターン（**片側交互通行 / 車線減少 / 通行止め+迂回**）を起点・終点の 2 クリックで**半自動配置**
- 保安施設の記号（ガードマン・カラーコーン・A 型バリケード・各種標示板/標識・作業車 ほか 20 種超）の手動配置
- **作業帯**（ピンク塗り＋斜線）、**導流帯**（ゼブラ斜線＋実線外周）、歩行者通路、コーン列、寸法線、文字の作図
- 使用した記号からの**凡例自動生成**
- 移動 / 回転 / 頂点編集 / 複製 / 削除 / Undo・Redo
- localStorage 自動保存、JSON 保存・読込（背景地図ごと復元）
- 2 倍解像度 PNG 出力、**A4 / A3 横 印刷**

### しない（意図的に対象外）
- ラスタ地形図からの**完全自動作図**（道路中心線・幅員・交通流の自動認識）。精度が許可申請の水準に達せず、警察協議で通らないため、半自動（利用者が骨格を指定）が実用上の最適点。
- 規制方法・保安施設の要否の**判断**。これは所轄警察・道路管理者との協議事項であり、ツールは作図補助に徹する。

## 2. 動作環境と既知の制約

- **ブラウザ**: Chrome / Edge 等のモダンブラウザ（デスクトップ）。Canvas 2D 前提。
- **PDF 読込は初回のみインターネット接続が必要**（PDF.js を CDN 取得）。オフライン環境では PDF を画像化してから読込む運用。エラー時はその旨を案内するダイアログを表示。
- **印刷**: 非表示 iframe に描画画像を流し込み `iframe.contentWindow.print()` を呼ぶ方式（ポップアップブロックの影響を受けない）。用紙サイズは印刷ダイアログ側でも要確認。
- **自動保存**は localStorage。容量超過やプライベートモードでは黙って失敗するため、**確実な保管は JSON 保存**を案内済み。背景画像を含むと dataURL が大きく、localStorage 上限（多くの環境で ~5MB）に当たり得る。
- 用紙は内部的に A3 相当のピクセル寸法（1587×1122px, √2:1 横）で描画。A4/A3 は同じ縦横比のため、印刷時に用紙へフィットさせるだけで両対応。

## 3. 基本操作フロー

1. **地形図読込 (PDF/画像)** — ヘッダー左のボタン。PDF は 1 ページ目を自動でラスタ化。
2. **縮尺設定** — ボタン押下後、地図上で距離の分かる 2 点をクリック → モーダルで実距離(m)を入力。
3. **規制パターン自動配置** — モーダルでパターン・作業帯幅・コーン間隔・設置側を指定 → 図面で起点→終点を 2 クリック。
4. **手直し** — 選択してドラッグ移動 / `R`=15°回転 / `Delete`=削除 / 頂点ドラッグで変形 / 右パネルで寸法・ラベル編集。
5. **出力** — PNG 出力 または 印刷（A4/A3 を選択）。

### キーボード / マウス
- 選択・移動: クリック / ドラッグ
- 回転: `R`（15°ずつ）、削除: `Delete` / `Backspace`
- Undo/Redo: `Ctrl+Z` / `Ctrl+Y`（`Ctrl+Shift+Z` も Redo）
- 画面移動: `Space`+ドラッグ / 右ドラッグ / 中ボタンドラッグ、ズーム: ホイール
- 線/多角形の確定: ダブルクリック または `Enter`、中止: `Esc`

## 4. データモデル（保存 JSON / 内部 state）

```
state = {
  objects: [ ...図形オブジェクト ],
  settings: {
    title, koshu,            // 図面名・工種
    showLegend, showNote,    // 凡例・注記の表示
    symScale,                // 記号の全体倍率
    mPerPx,                  // 縮尺: 1世界px あたりのメートル数（既定 0.05 = 1px:5cm）
    calibrated,              // 縮尺設定済みフラグ
    bgOpacity,               // 背景透明度
    legendPos,               // 凡例の位置（null=右下自動）
  },
  bg: { img, dataURL, x, y, w, h } | null,   // 背景地図（保存時は dataURL と矩形のみ）
  view: { z, ox, oy },       // 画面のズーム・平行移動（保存対象外）
  sel, tool, draft, undoStack, redoStack,    // 実行時状態
}
```

### オブジェクト型（`objects[]` の要素）
| type | 内容 | 主なプロパティ |
|---|---|---|
| `symbol` | 記号 1 個 | `kind`(KINDS のキー), `x, y, rot, scl` |
| `zone` | 作業帯（工事箇所） | `pts[]`（多角形頂点） |
| `divzone` | **導流帯**（v1.1 追加） | `pts[]`, `stripe`(斜線間隔 m), `fill`(内部薄塗り), `color` |
| `pedpath` | 歩行者通路 | `pts[]`, `width`(m) |
| `conerow` | コーン列 | `pts[]`, `interval`(m), `bar`(コーンバー有無) |
| `dim` | 寸法線 | `a, b`(端点), `off`(引出し量), `text`(空欄=実測値自動) |
| `text` | 文字 | `x, y, text, size, rot` |

座標は「世界座標(px)」。実寸は `px * settings.mPerPx` で算出。

## 5. 規制パターン自動配置の中身（runTemplate）

`runTemplate(A, B)` は起点 A・終点 B と `wizardParams`（pattern / width / cone / taper / side）から部材を一括生成する。

- **katagawa（片側交互通行）**: 作業帯 + 交通側コーン列 + 両端にガードマン・片側交互通行標示板・停止位置標示板・矢印板 + 工事看板 + 作業車。
- **shasen（車線減少）**: 作業帯 + 進行方向手前のテーパー（すり付け）コーン列 + 車線減少標示板・幅員減少看板・矢印板。
- **tome（通行止め+迂回）**: 全幅を作業帯化 + 両端に A 型バリケード 3 基横断 + 通行止め標示板・迂回路標示板・ガードマン・工事看板。

生成後の部材はすべて個別に移動・回転・削除できる（テンプレートは初期配置の足場に過ぎない）。

---

## 6. 出力・印刷

- **PNG**: `renderExportCanvas(2)` で 2 倍解像度の画面外 Canvas に `drawSheet` を再描画し、`toBlob` でダウンロード。
- **印刷**: `doPrint('A4'|'A3')` が同じ 2 倍画像を非表示 iframe に `<img>` として書き込み、`@page{size:… landscape}` を当てて `contentWindow.print()`。ポップアップに依存しない。

---

## 7. 免責・運用上の注意（UI 内「使い方」にも明記）

- 本ツールは**作図補助**。規制方法・保安施設の要否・配置基準は所轄警察および道路管理者との協議による。
- 自動配置は定型の初期骨格であり、そのまま提出可能な図面を保証しない。必ず目視確認・手直しを行う。
